# 📖 Quick Comp Playbook — the business, documented

This folder is the half of the business that is NOT code: how a launch works,
who does what, the scripts, the GHL recipes, the env-var catalog and how to
back everything up. The other half (the engine) is described in the repo-root
`CLAUDE.md`.

Quick Comp runs on the same **business operating system** as ALTO Pro (the
contractor product). ALTO is the master of the shared patterns; when ALTO
gains a fix or a play, port the PATTERN here (adapted to realtors, English
first) — never copy files wholesale.

Golden rule: **the engine is one; what changes per brand is a copy pack**
(vocabulary, prices, demos, ads). If anything in this playbook goes stale,
update it in the same commit that changed the system.
